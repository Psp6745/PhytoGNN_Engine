## 📖 Overview

**PhytoGNN_Engine** is an AI-powered biomedical research platform designed to accelerate phytochemical drug discovery.

The platform transforms medicinal plants, phytochemicals and diabetes-related proteins into an intelligent knowledge graph and applies Graph Neural Network concepts to discover and visualize meaningful biological relationships.

It provides an interactive web application where researchers, students and healthcare enthusiasts can explore plants, active compounds, protein targets, AI predictions and graph-based discoveries.
